# Todo App
A simple Todo App built with React
## Features

- Add new tasks
- Mark tasks as completed
- delete tasks
- view completed tasks
- view active tasks
## Technologies

- React
- Javascript
- CSS
- HTML
## Installation

1. Clone the repository
```bash
git clone git@github.com:keitamady/todo.git
```
2. Go to the project folder
```bash
cd three
```
3. Install dependencies
```bash
npm install
```
4. Start the development server
```bash
npm run dev
```
